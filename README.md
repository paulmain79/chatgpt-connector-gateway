# ChatGPT Connector Gateway

Give ChatGPT arms and legs! Connect to GitHub, Vercel, Google Sheets, and more.

## Deploy to Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fpaulmain79%2Fchatgpt-connector-gateway&env=GATEWAY_KEY,GITHUB_TOKEN,VERCEL_PREVIEW_HOOK,VERCEL_PROD_HOOK&envDescription=API%20keys%20needed%20for%20the%20gateway&project-name=chatgpt-gateway)

## Quick Setup

1. Click deploy button above
2. Add your environment variables:
   - GATEWAY_KEY: Your secret password
   - GITHUB_TOKEN: From github.com/settings/tokens
   - VERCEL_PREVIEW_HOOK: From Vercel project settings
   - VERCEL_PROD_HOOK: From Vercel project settings

3. Give the endpoint to ChatGPT with instructions

## Usage

Tell ChatGPT it has access to your gateway and can:
- Deploy to Vercel
- Create GitHub issues
- Run GitHub workflows
- Update files
- Query databases
